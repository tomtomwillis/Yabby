What is YabbyVille?

Yabbyville is a self hosted, community focussed, music streaming platform. It’s essentially like Spotify, except we own all the music and can upload and listen to whatever we like. It’s basically all hosted on a server at Hamish’s house and luckily isn’t helping to invest $600 million in Israeli weapons. We wanted to make this for a few reasons but would love to give you another push to delete your spotify accounts.

A more technical explanation is that it uses the open source software Navidrome, and the Subsonic API, to make a database of music you can listen to anywhere. We’ve created a front end website called YabbyVille which brings together some additional parts that add extra functionality - like a message board, a sticker function to recommend music to other users, a portal for uploading music and a software called Slskd that lets you torrent music to the server.

In the back end we are also using a service called beets to manage the music library and keep it neat and tidy. The website is built in React. All of this is hosted on Hamish’s server (RAID 1 NAS) with the exception of some features that use a Firebase database (boo google but we are using a free tier until we can self host) and a Storj Bucket. We hope eventually to become fully self-hosted.

List of Log Ins needed

To access all the parts of YabbyVille you will need:

  1. Email/Password for the Yabbyville site
  2. Username/Password for Navidrome
  3. Username/Password for Slskd

1. YabbyVille Log In

Once you give Hamish/Tom an Email address they can make an account for you. They will then send you an email to reset your password.

This will let you access the YabbyVille Homepage and other parts of the site like the message board. Don’t forget to go to the Profile Page to add a username and avatar.

2. Navidrome Log In

This is where all the music is. Each user has their own log in details.

Like before you need to provide us with a username you want. We will then give you a temporary password which you can reset yourself later.

This will let you access Navidrome on the browser and is what you will use to log into your navidrome client if you want to use one to listen to music on your phone. See below for more info about this.

3. Slskd Log In

This is the log in details for a service called slskd. It lets you download files from Soulseek so you can add them to the server automatically. See below for more details. The username/password is the same for everyone on this. Message Tom/Hamish for this but make sure you keep it very safe to keep the server secure.

How do I listen to music?

Listening in Navidrome

Navidrome is the service that is used to access all the music we have saved. You can get to Navidrome here, or by clicking the Listen button on the YabbyVille homepage. Once you log in with your username and password you can listen to any music you would like in the library on the navidrome web browser.

Listening on your phone/device using a Navidrome Client

If you would like to listen to Navidrome from another device (like your phone) you will need to download a Navidrome Client. This is an app that connects to Navidrome and gives you a nice user interface, lets you download tracks to your phone and all the best bits of using an app like spotify.

There is a full list of all the Navidrome clients on the navidrome website here but any Subsonic compatible client should work.

When you open these apps they will ask for your:

  • Server Address: https://music.yabbyville.xyz/ 
  • It may request Port Number which is 443
  • Username: your navidrome username
  • Password: your navidrome password

Warning ⚠️ - file sizes

Please note that a lot of the library is FLAC files! These sound great, but are about 2-3x the size of 320kbps mp3. This can use a lot more data than expected while streaming. We do not currently support server-side transcoding. A Spotify subscription currently costs £12, if you invest half of that into upgrading your phone plan you are unlikely to have any issues. Most players also supported locally downloading files for offline playback.

We’re considering converting the library entirely to mp3, to reduce space and streaming requirements. We’re also considering enabling server-side transcoding. Get in touch if you have any opinions about this.

How do I add new music to YabbyVille/Navidrome?

Before adding any music please try to make sure that anything you are uploading or downloading is of good sound quality and has the correct metadata when possible. We prefer 320 kbps MP3 files or FLAC. The system will try to add album titles or missing metadata but it won’t always get it right, and the better foundation you give it the more likely it is to match correctly. We use Musicbrainz as a metadata source, if your music isn’t in there please add it! There are some known issues with playing M4A files - not a lot of browsers support these.

There are two ways to add music to the server so that you and others can listen to it.

  1. Add your own music files from your laptop
  2. Download/Torrent music from the internet that will get automatically added to the server

Upload your own files: [please note that this feature isn’t currently working]

  • Start on the YabbyVille homepage and click the button titled Upload
  • You will be able to drag and drop all the files/folders you want to in the window. Click the upload button to begin uploading to the server.
  • Please note that there is a limit of 5GB of files at once and that no single file can be larger than 500mb.
  • Files should have good metadata - this is how Navidrome sorts and structures our library, so please don’t upload junk. Beets uses the Musicbrainz database, so if you want to be good, run your music through Musicbrainz Picard first - and submit a release for those items that don’t have one!

Use slskd to download music files to be added to the server

If you are unfamiliar with Soulseek or slskd there is a good explanation on their github here. Slskd is an application that uses Soulseek to make it easy to download files from other Soulseek users. There is no risk to your device when using slskd as all the work is being done on the server that hosts YabbyVille.

  • Start on the YabbyVille homepage and click the button titled Request.
  • You will need the user/pw to log into slskd. See the Getting Started section.
  • Search for albums you would like to download. You can either select the individual file you want to download, or if you click at the top of the folder you can select the entire folder.
  • You can track the download progress from the download tab

Scanning new files in Navidrome

Once music is up/downloaded to the server it will be automatically processed by the server to try and make sure it has the correct cover art/name. It will then need to be processed by Navidrome to make it visible in the recently added section.

It can take a min or two to complete the download and process the files completely, but if your files don’t turn up, they might need to be manually processed and have metadata associated with them.

Previously, you needed to trigger a library scan to have music show up. Navidrome is pretty good at detecting changes on it’s own now, but if you need to, you can trigger a scan manually. To trigger a scan you need to open Navidrome and click on the refresh icon in the top right corner 🔄.

What are the stickers?

Stickers are a way for you to customise your account. You can also add your sticker to albums on the YabbyVille homepage with a message to recommend them to other users/say something about them. The idea is like in a record shops when staff will add a staff recommended sticker onto an album.

Customising your Sticker

You can customise your personal sticker and your username in the profile page.

You can change your sticker whenever you like but the sticker you had when you added one to an album will remain the same.

Please send any suggestions for any new stickers we could add in!

Adding Stickers to Albums

When you load the YabbyVille homepage you can see a carousel of stickers left on albums by other YabbyVille users. Click on one to see what is being said or to listen to the album in question. Go to the stickers page to see all the user left stickers.

To add a sticker to an album:

  • Open Navidrome and navigate to the page for the album you want to recommend.
  • Copy the URL - it should look something like this https://music.yabbyville.xyz/app/#/album/34Srzyl7KA2EFTrqhLwt82/show 
  • Go to the YabbyVille homepage and paste the URL into the Place a Sticker form, then click Fetch.
  • This should fetch the album cover for you. Click on it and drag to place your sticker.
  • Type your comment and click post.
  • You may need to refresh the page before your sticker is visible.

Where you place the sticker should be where the sticker remains when it is on the homepage. You can add as many stickers from as many users as you like to albums.
