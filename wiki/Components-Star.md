# Star

**File:** `src/components/basic/Star.tsx`

An animated SVG star used as a background decoration on the app. Uses an `feTurbulence` SVG filter to create a stop-motion distortion effect.

Rendered once in `App.tsx` and visible across all pages.

## Usage

```tsx
<Star />
```

No props. To remove it from the app, delete the `<Star />` line in `App.tsx`.
